##  Common Emmet Abbreviations

1. `!` → Basic HTML5 template
2. `div` → `<div></div>`
3. `div.classname` → `<div class="classname"></div>`
4. `#idname` → `<div id="idname"></div>`
5. `ul>li*3` → Creates a `<ul>` with 3 `<li>` items.
6. `a:link` → `<a href=""></a>`
7. `img` → `<img src="" alt="" />`
8. `p{Hello World}` → `<p>Hello World</p>`
