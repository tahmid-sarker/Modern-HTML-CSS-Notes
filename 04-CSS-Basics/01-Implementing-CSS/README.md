## Overview

1. **Inline CSS** → Added directly in the element. Quick but hard to maintain.
2. **Internal CSS** → Inside `<style>` in `<head>`. Good for small pages.
3. **External CSS** → Separate `.css` file linked with `<link>`. Clean and reusable.
